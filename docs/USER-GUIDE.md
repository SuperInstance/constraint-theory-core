## 1. Lattice ŌĆö Eisenstein AŌéé Quantization

The AŌéé lattice is generated by {1, Žē} where Žē = e^(2ŽĆi/3). Every point in the plane is within Žü = 1/ŌłÜ3 of a lattice point. This isn't an approximation ŌĆö it's a geometric guarantee.

### A2Point

An immutable point on the lattice, represented as `(a, b)` meaning a┬Ę1 + b┬ĘŽē.

```python
from constraint_theory_core.lattice import A2Point

p = A2Point(3, -1)

# Arithmetic
q = A2Point(1, 2)
print(p + q)        # A2Point(a=4, b=1)
print(p - q)        # A2Point(a=2, b=-3)
print(-p)           # A2Point(a=-3, b=1)

# Squared Eisenstein norm: a┬▓ - ab + b┬▓
print(p.norm_sq())  # 3┬▓ - 3(-1) + (-1)┬▓ = 13

# Convert to Cartesian coordinates
x, y = p.to_complex()
print(f"({x:.4f}, {y:.4f})")  # (3.5000, -0.8660)
```

**Why a┬▓ ŌłÆ ab + b┬▓?** Because in the complex plane, z = a + bŽē where Žē = ŌłÆ┬Į + iŌłÜ3/2. The norm |z|┬▓ = a┬▓ + ab┬Ę(ŌłÆ1) + b┬▓ = a┬▓ ŌłÆ ab + b┬▓. Always non-negative.

### snap(x, y) ŌåÆ (A2Point, error)

Snap any point to the nearest lattice point.

```python
from constraint_theory_core.lattice import snap

# Origin snaps exactly
pt, err = snap(0.0, 0.0)
print(pt)    # A2Point(a=0, b=0)
print(err)   # 0.0

# A point near (0, 1) in lattice coords
pt, err = snap(-0.5, 0.866)  # Ōēł Žē
print(pt)    # A2Point(a=0, b=1)
print(err)   # ~0.0

# Random point ŌĆö error is always Ōēż Žü
pt, err = snap(0.73, -0.42)
print(f"Error: {err:.4f}")  # Something Ōēż 0.577
```

### covering_radius() and is_safe()

```python
from constraint_theory_core.lattice import snap, covering_radius, is_safe

rho = covering_radius()  # 1/ŌłÜ3 Ōēł 0.5774

pt, err = snap(0.3, 0.2)
print(f"Error {err:.4f} Ōēż Žü = {rho:.4f}: {err <= rho}")  # Always True

# is_safe checks err < Žü/2 Ōēł 0.289
# If safe, no other lattice point could be closer
print(f"Safe snap: {is_safe(err)}")
```

### Dodecet ŌĆö 12 Nearest-Neighbor Directions

The AŌéé lattice has exactly 12 minimal nonzero vectors. Encode/decode them:

```python
from constraint_theory_core.lattice import decode_dodecet, encode_dodecet

# Decode index ŌåÆ direction vector
d = decode_dodecet(0)  # A2Point(a=1, b=0)
print(d)

# Encode direction ŌåÆ index
idx = encode_dodecet(d)  # 0
print(idx)

# Roundtrip all 12
for i in range(12):
    assert encode_dodecet(decode_dodecet(i)) == i
```

The 12 directions are: `(1,0), (0,1), (ŌłÆ1,1), (ŌłÆ1,0), (0,ŌłÆ1), (1,ŌłÆ1), (2,ŌłÆ1), (1,ŌłÆ2), (ŌłÆ1,ŌłÆ1), (ŌłÆ2,1), (ŌłÆ1,2), (1,1)`.

### Vector48 ŌĆö 48 Pythagorean Directions

Angles quantized to 48 directions (7.5┬░ apart, 360┬░/48). Used for holonomy verification.

```python
from constraint_theory_core.lattice import vector48_encode, vector48_decode
import math

# Encode an angle ŌåÆ index 0-47
idx = vector48_encode(math.pi / 4)   # ~index 6 (45┬░ / 7.5┬░)
print(idx)

# Decode back
angle = vector48_decode(idx)
print(f"{math.degrees(angle):.1f}┬░")

# Roundtrip
for i in range(48):
    assert vector48_encode(vector48_decode(i)) == i
```

### Holonomy Product (Lattice-Level)

```python
from constraint_theory_core.lattice import holonomy_product, is_consistent

# 12+12+12+12 = 48 ŌēĪ 0 mod 48 ŌåÆ consistent cycle

# 1+2+3 = 6 ŌēĀ 0 mod 48 ŌåÆ inconsistent
```

## 2. Temporal ŌĆö Deadband Funnel

The temporal module wraps lattice quantization in an exponentially decaying deadband. Think of it as a funnel: wide at first (permissive), narrowing over time (precise). If the error exceeds the funnel, that's an anomaly.

### TemporalAgent

```python
from constraint_theory_core.temporal import TemporalAgent, FunnelPhase

agent = TemporalAgent(
    decay_rate=0.1,    # ╬╗ in ╬Ą(t) = ╬ĄŌéĆ ┬Ę e^(-╬╗t)
    epsilon_0=0.5,     # initial deadband width
    delta=0.577,       # anomaly threshold (absolute)
)

print(f"Initial ╬Ą = {agent.epsilon:.4f}")  # 0.5000
```

### Observing Points

```python
# Observe a point near the origin ŌĆö error is small
result = agent.observe(0.01, 0.01, t=1.0)
print(result.phase)       # FunnelPhase.NARROWING
print(result.error)       # ~0.01
print(result.deadband)    # Slightly less than 0.5 (decayed)
print(result.snapped_a)   # 0
print(result.snapped_b)   # 0
```

### Three Phases

```python
# NARROWING: error < ╬Ą, funnel tightens
result = agent.observe(0.01, 0.01, t=2.0)
assert result.phase == FunnelPhase.NARROWING

# APPROACH: ╬Ą < error < ╬┤, not yet anomalous but outside deadband
# (happens when deadband has narrowed past the error)

# ANOMALY: error > ╬┤, funnel resets to full width
result = agent.observe(10.0, 10.0, t=3.0)
assert result.phase == FunnelPhase.ANOMALY
print(f"Anomalies: {agent.anomaly_count}")  # 1
```

### Convergence Over Time

```python
agent = TemporalAgent(decay_rate=0.5, epsilon_0=0.5)

for t in range(1, 100):
    result = agent.observe(0.0, 0.0, t=float(t))

print(f"Final ╬Ą = {agent.epsilon:.6f}")  # ~0.0000 (narrowed to near-zero)
```

### Manual Control

```python
# Manual decay
agent.decay(dt=2.0)

# Reset to initial width
agent.reset(0, 0)
assert agent.epsilon == 0.5
```

## 3. Rigidity ŌĆö Laman Graphs

A graph G = (V, E) is **Laman rigid** iff:
1. |E| = 2|V| ŌłÆ 3 (exact edge count)
2. Every subset of k vertices has Ōēż 2k ŌłÆ 3 edges (no subgraph is over-constrained)

This means: every edge is load-bearing, removing any edge loses rigidity, and there are Ōēź2 independent paths between any two vertices.

### is_laman(n, edges)

```python
from constraint_theory_core.rigidity import is_laman

# KŌéé (2 vertices, 1 edge) ŌĆö the simplest Laman graph
print(is_laman(2, [(0, 1)]))  # True

# KŌéā (3 vertices, 3 edges) ŌĆö 2┬Ę3-3 = 3 Ō£ō
print(is_laman(3, [(0,1), (1,2), (0,2)]))  # True

# Too few edges for n=4 (need 2┬Ę4-3=5)
print(is_laman(4, [(0,1), (1,2), (0,2)]))  # False

# KŌéä has 6 edges > 2┬Ę4-3=5 ŌĆö not Laman
print(is_laman(4, [(0,1),(1,2),(2,3),(3,0),(0,2),(1,3)]))  # False
```

### henneberg_construct(n, seed=42)

Build a Laman graph via Henneberg type-I construction: start with KŌéé, then each new vertex connects to 2 existing vertices.

```python
from constraint_theory_core.rigidity import henneberg_construct

# 5 vertices ŌåÆ 2┬Ę5-3 = 7 edges
edges = henneberg_construct(5)
print(f"Edges: {len(edges)}")  # 7
print(f"Is Laman: {is_laman(5, edges)}")  # True

# 9 agents for a fleet
edges = henneberg_construct(9)
print(f"Edges: {len(edges)}")  # 15 (= 2┬Ę9-3)
```

Reproducible with seed:
```python
e1 = henneberg_construct(20, seed=42)
e2 = henneberg_construct(20, seed=42)
assert e1 == e2
```

### algebraic_connectivity(edges, n)

Computes ╬╗Ōéé, the second-smallest eigenvalue of the graph Laplacian. This determines:
- **╬╗Ōéé > 0** ŌåÆ graph is connected
- **╬╗Ōéé value** ŌåÆ convergence rate of consensus (higher = faster)
- **Optimal coupling** ŌåÆ ╬▒* = 2/(╬╗Ōéé + ╬╗ŌéÖ)

```python
from constraint_theory_core.rigidity import algebraic_connectivity

# Simple 2-node graph
print(algebraic_connectivity([(0,1)], 2))  # ~2.0

# Laman graph for 9 agents
edges = henneberg_construct(9)
lam2 = algebraic_connectivity(edges, 9)
print(f"╬╗Ōéé = {lam2:.4f}")  # Positive (graph is connected)
```

### optimal_coupling(edges, n)

Computes the optimal correction strength for consensus: ╬▒* = 2/(╬╗Ōéé + ╬╗ŌéÖ).

```python
from constraint_theory_core.rigidity import optimal_coupling

edges = henneberg_construct(9)
alpha = optimal_coupling(edges, 9)
print(f"╬▒* = {alpha:.4f}")  # Between 0 and 1
```

- Too small ╬▒ ŌåÆ slow convergence
- Too large ╬▒ ŌåÆ overshoot and oscillation
- ╬▒* ŌåÆ fastest convergence without oscillation

## 4. Metronome ŌĆö Distributed Consensus

The metronome drives consensus across Laman-connected agents. Each agent maintains a phase Žå Ōłł [0, 2ŽĆ) and corrects toward neighbors using the optimal coupling ╬▒*.

### Creating Agents

```python
from constraint_theory_core import Metronome, henneberg_construct
import math

# Build topology for 9 agents
edges = henneberg_construct(9)

# Helper to find neighbors of agent i
def neighbors(edges, i):
    return [v if u == i else u for u, v in edges if u == i or v == i]

# Create agents with spread-out initial phases
agents = [
    Metronome(
        T=1.0,                          # 1 second per tick
        phi0=math.pi * i / 9,           # spread phases
        epsilon=0.5,                    # deadband tolerance
        delta=0.6,                      # anomaly threshold
        neighbors=neighbors(edges, i),  # Laman neighbors
        edges=edges,                    # full edge list for ╬▒*
        n_agents=9,                     # total agents
    )
    for i in range(9)
]
```

### Tick, Agree, Correct

```python
# One round of consensus
for agent in agents:
    agent.tick()                        # advance phase by one period

# Exchange phases with neighbors
phases = [a.phase for a in agents]
for agent in agents:
    neighbor_phases = [phases[j] for j in agent.neighbors]
    correction = agent.correct(neighbor_phases)

# Check pairwise agreement
m1, m2 = agents[0], agents[1]
print(m1.agree(m2))  # True if phase difference Ōēż ╬Ą
```

### Checking State

```python
state = agents[0].state()
print(state.phase)          # current phase in radians
print(state.tick_count)     # ticks elapsed
print(state.converged)      # True if 3 consecutive tiny corrections
print(state.epsilon)        # current deadband width
print(state.anomaly_count)  # anomalies detected
```

### Observing External Reference

```python
# Snap to an external reference point
phase = agents[0].observe(1.0, 0.0)  # observe point (1, 0)
# Phase snaps to lattice angle of (1, 0) ŌåÆ 0 radians
```

### Reset

```python
agents[0].reset()  # back to initial phase, clears corrections
```

## 5. Holonomy ŌĆö Cycle Verification

In a tiled constraint system (like PLATO), each tile is a cycle. A cycle has **zero holonomy** when its direction vectors sum to 0 mod 48 ŌĆö meaning the cycle closes exactly. Non-zero holonomy means inconsistency.

### cycle_holonomy(edges, directions)

```python
from constraint_theory_core.holonomy import cycle_holonomy

tri = [(0,1), (1,2), (2,0)]

# 16 + 16 + 16 = 48 ŌēĪ 0 mod 48 ŌåÆ consistent
print(cycle_holonomy(tri, [16, 16, 16]))  # 0

# 1 + 2 + 3 = 6 ŌēĀ 0 ŌåÆ inconsistent
print(cycle_holonomy(tri, [1, 2, 3]))     # 6
```

### verify_consistency(tiles)

Check all tiles at once:

```python
from constraint_theory_core.holonomy import verify_consistency

tri = [(0,1), (1,2), (2,0)]
tri2 = [(0,1), (1,3), (3,0)]

tiles = [
    (tri, [16, 16, 16]),     # consistent
    (tri2, [16, 16, 16]),    # consistent
]
print(verify_consistency(tiles))  # True

tiles.append((tri, [1, 2, 3]))   # inconsistent!
print(verify_consistency(tiles))  # False
```

### isolate_fault(tiles) ŌĆö O(log N)

Find an inconsistent tile via binary bisection:

```python
from constraint_theory_core.holonomy import isolate_fault

good = ([(0,1),(1,2),(2,0)], [16,16,16])
bad = ([(0,1),(1,2),(2,0)], [1,2,3])

tiles = [good] * 50 + [bad] + [good] * 49  # 100 tiles, fault at index 50
print(isolate_fault(tiles))  # 50
```

This runs O(log N) consistency checks instead of O(N).

### fault_boundaries(tiles) ŌĆö Find All Faults

```python
from constraint_theory_core.holonomy import fault_boundaries

tiles = [good, bad, good, bad, bad]
print(fault_boundaries(tiles))  # [1, 3, 4]
```

Linear scan ŌĆö O(N) ŌĆö but finds every inconsistent tile.

## Putting It All Together

The five modules compose into a complete constraint-theory pipeline:

```python
from constraint_theory_core import *
from constraint_theory_core.holonomy import verify_consistency, isolate_fault
import math

# 1. Build Laman topology for 9 agents
edges = henneberg_construct(9)

# 2. Create metronome agents
def neighbors(edges, i):
    return [v if u == i else u for u, v in edges if u == i or v == i]

agents = [
    Metronome(T=1.0, phi0=math.pi * i / 9,
              epsilon=0.5, delta=0.6,
              neighbors=neighbors(edges, i),
              edges=edges, n_agents=9)
    for i in range(9)
]

# 3. Run consensus for 50 ticks
for _ in range(50):
    for a in agents:
        a.tick()
    phases = [a.phase for a in agents]
    for a in agents:
        nbr_phases = [phases[j] for j in a.neighbors]
        a.correct(nbr_phases)

# 4. Verify holonomy of the cycle structure
tiles = [
    ([(i, neighbors(edges, i)[0]), 
      (neighbors(edges, i)[0], neighbors(edges, i)[1]),
      (neighbors(edges, i)[1], i)],
     [16, 16, 16])
    for i in range(min(3, len(edges)))
]
print(f"Cycles consistent: {verify_consistency(tiles)}")
print(f"Agents converged: {all(a.converged for a in agents)}")
```
